# Minimal React + Webpack + Babel + Mirador 2 setup

Based on [the Minimal React Webpack Babel Setup](https://www.robinwieruch.de/minimal-react-webpack-babel-setup/)

## Features

* React 16
* Webpack 4
* Babel 7
* Hot Module Replacement
* HTML Webpack Plugin
* Mirador 2

## Development

* yarn start
* visit `http://localhost:8080/`
